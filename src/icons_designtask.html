Design Task: Creating an icon to represent a pen-drive.

Pen-Drives come in different shapes, sizes, colors in the real world. Gone are the days when the save icon was a diskette. Increasing popularity of pen-drives nessiciates a specific icon which is different from hard disk drives and other saving devices. Design graphically one such icon that can be used by college going students on their computer using the procedure given earlier under the PROCEDURE tab.

To help you achieve this task a copy of the INKSCAPE tool can be downloaded.
